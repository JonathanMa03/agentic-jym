Project: Predictors of Academic Performance: The Role of Alcohol Consumption and Gender
Description: This project investigates whether alcohol use predicts academic performance and whether gender moderates the relationship. The analysis uses linear regression with interaction terms and controls for more than 30 covariates.
Technologies: R, dplyr, ggplot2

Project: Obesity of the Youth: Health Disparity and Income Inequality
Description: This project studies adolescent obesity using logistic models and inequality indices on NSCH data. It explores the relationship between obesity, income, race, and healthcare access.
Technologies: R, ineq, tidyverse, lme4, dplyr, ggplot2

Project: Stock Price Forecasting with Neural Networks
Description: This project is a Dash-based web application for real-time stock prediction using neural networks and interactive visualizations.
Technologies: Python, plotly, numpy, pandas, dash

Project: Predicting Depression: A Comparative ML Approach
Description: This project compares logistic regression and random forest models to detect depression using demographic and psychosocial data.
Technologies: R, CARET, dplyr, randomForest
